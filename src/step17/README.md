# Step 17: 旅行小管家

基于 Step 16 的 Agent 状态与记忆系统，构建智能旅行规划助手。

## 项目概述

**旅行小管家**是一个懂你的智能旅行助手：
- ✈️ 根据预算、时间、喜好推荐目的地
- 🏨 查询实时天气、机票、酒店信息
- 🗺️ 生成个性化行程（景点、餐厅、交通）
- 📝 自动创建行李清单和备忘提醒
- 💬 像朋友一样聊天，记得你的旅行偏好

## 新增文件

1. `include/travel/` - 旅行相关模块
   - `trip_planner.hpp` - 行程规划器
   - `destination_db.hpp` - 目的地数据库
   - `budget_calculator.hpp` - 预算计算器
2. `tools/travel/` - 旅行工具
   - `weather_tool.hpp` - 天气查询
   - `flight_tool.hpp` - 航班搜索
   - `hotel_tool.hpp` - 酒店比价
   - `attraction_tool.hpp` - 景点推荐

## 使用示例

```
用户：我想下个月去日本看樱花，预算 1 万，5 天时间

小管家：🌸 樱花季！好选择！让我帮你规划一下...

✈️ 推荐路线：上海 → 大阪 → 京都 → 东京
📅 最佳时间：3 月底 - 4 月初
💰 预算分配：
   - 机票：3000-4000
   - 住宿：2500
   - 交通：800
   - 餐饮：2000
   - 门票购物：1700
```
