# Step 18: 虚拟咖啡厅「星语轩」

基于 Step 16 的 Agent 状态与记忆系统，构建多 AI 角色共存的虚拟社交空间。

## 项目概述

欢迎来到 **「星语轩」** —— 一家位于虚拟街角的温馨咖啡厅。

### 角色设定

- 👨‍🍳 **老王** —— 42岁咖啡师，沉默寡言但手艺精湛
- 🎨 **小雨** —— 26岁插画师，活泼开朗，在窗边画画
- 💻 **阿杰** —— 30岁程序员，暗恋小雨
- 📚 **林阿姨** —— 58岁退休教师，像妈妈一样
- 🐱 **橘子** —— 咖啡厅的猫

### 核心特性

- 每个角色有自己的情绪、记忆、关系网
- NPC 之间会自发互动、聊天、甚至争吵
- 玩家可以点咖啡、听他们聊天、影响故事走向
- 关系网络可视化，见证角色间的羁绊

## 新增文件

1. `include/cafe/` - 咖啡厅系统
   - `npc_agent.hpp` - NPC Agent 基类
   - `world_state.hpp` - 世界状态管理
   - `social_event.hpp` - 社交事件系统
   - `relationship_graph.hpp` - 关系网络
2. `npcs/` - 角色定义
   - `laowang.hpp` - 老王
   - `xiaoyu.hpp` - 小雨
   - `ajie.hpp` - 阿杰
   - `linaiyi.hpp` - 林阿姨
